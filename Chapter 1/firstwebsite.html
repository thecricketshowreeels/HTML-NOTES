Chapter 1 - Creating First Website

We start building a website by creating a file named index.html. index.html is a special filename that is presented when the website root address is typed.

A Basic HTML Page (Boiler Plate Code)

<!DOCTYPE html>        → Specifies this is an HTML5 document 
<html>        → Root of an HTML page 
    <head>        → Contains page metadata 
        <title>Document</title>        → Contains title 
    </head>        → Head closing tag
    <body>        → The main body of the page (rendered by browser) 
        <h1> Heading </h1>        → Heading 
    </body>        → Body closing tag
</html>        → HTML closing tag

Visualization of an HTML Document

A tag is like a container for either content or other HTML tags. Opening Tag + Content + Closing Tag = HTML Element.

Important Notes

  • HTML is the parent of Head and Body tags.
  • Most HTML elements have opening and closing tags with content in between.
  • Some HTML tags have no content; these are called empty elements, e.g., <br>.
  • We can either use .htm or .html extension.
  • You can use "Inspect Element" or "View Page Source" in Chrome to look into a website's HTML code.

Comments In HTML

Comments in HTML are useful for adding notes or explanations in the code. They are ignored by the browser. 

<!-- This is a comment -->

Case Sensitivity

HTML is not a case-sensitive language. <H1> and <h1> tags are the same.

Container and Empty Tags In HTML

1. Container Tags (Paired Tags)

  • Opening Tag: The opening tag starts with < and ends with >. For example, <p>.
  • Closing Tag: The closing tag also starts with < but includes a forward slash / before the tag name, and ends with >. For example, </p>.

Examples:

  • Paragraph: 
    <p>This is a paragraph.</p>
  • Heading: 
    <h1>This is a heading.</h1>

2. Empty Tags (Unpaired Tags)

  • Self-Closing Tag: A self-closing tag starts with < and ends with />. For example, <img /> or <br>.

Examples of self-closing tags:

  • Line Break: 
    <br>
  • Horizontal Rule: 
    <hr>
  • Image: 
    <img src="image.jpg" alt="An example image">

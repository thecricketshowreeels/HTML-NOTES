Chapter 0 - Introduction To HTML

What Is HTML

  • HTML → Hyper Text Markup Language
  • HTML is the language of the web, used to create websites.
  • We use HTML tags to define the look and feel of a website.
  • HTML defines the barebone structure or layout of web pages.
  • An HTML document typically has a .html or .htm extension and contains various elements defined by tags.
  • HTML5 is the most recent version of HTML.
  • With an understanding of these tags, we can create beautiful websites easily.

Features Of HTML

  • HTML is supported by all web browsers, ensuring consistency across platforms.
  • Supports images, videos, and audio (e.g., YouTube shows videos on their website).
  • HTML is a markup language, not a programming language.
  • HTML can be enhanced with other languages like CSS for styling and JavaScript for interactivity etc.
